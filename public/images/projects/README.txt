Optional project thumbnails for the Projects section.

Cards render perfectly without images. To add a thumbnail:

  1. Place an image here (16:9 aspect ratio recommended, e.g. 1280x720).
  2. In src/data/projects.js, uncomment / set the `image` field of the
     project, e.g.:  image: "images/projects/plantefy.png"

If an image file is missing, the card simply renders without it.
